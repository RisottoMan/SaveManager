# SaveManager Mod
Allows you to save your position and teleport back:

<img src="Photo/How%20it%20works.gif" alt="Gif">

### Settings
To change the settings, go to the Mod Menu:

<img width="334" height="300" alt="image" src="https://github.com/user-attachments/assets/6e102099-0ef2-4f79-8b47-4dfd41497e53" />

Click on the SaveManager button:

<img width="381" height="350" alt="image" src="https://github.com/user-attachments/assets/119dd8db-e4e2-47b1-8984-3fc1081222c1" />

### Buttons
- Save position - Default keybind is `F1`.
- Load position - Default keybind is `F2`.

<img width="425" height="318" alt="image" src="https://github.com/user-attachments/assets/b95fa3fa-fe7a-4b91-80db-e79b5b73bd60" />

## Install
This mod is used with [MelonLoader](https://github.com/LavaGang/MelonLoader).\
Mod dependency: [BBModMenu](https://github.com/MiaouZart/BBModMenu).
